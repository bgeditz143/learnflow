# 🤖 LearnFlow — HR Training Platform

AI-powered corporate training platform with Avatar Trainer.

---

## 🚀 Run Karne Ka Tarika

### Step 1 — Node.js Install Karo
Download karo: https://nodejs.org (LTS version)

### Step 2 — Folder Open Karo
```
Is zip file ko extract karo
learnflow folder mein jaao
```

### Step 3 — Terminal/Command Prompt mein ye commands chalaao

```bash
npm install
npm run dev
```

### Step 4 — Browser mein kholna
```
http://localhost:5173
```

---

## 📋 Features

- **Admin Panel** — Trainings create/assign/manage, Employee management, Reports, Leaderboard, Settings
- **Employee Panel** — My Trainings, AI Avatar Training Room, Assessment, Certificates, Rewards & Badges
- **AI Avatar Trainer** — Slide-by-slide training with live Q&A simulation
- **Assessment System** — MCQ + Pass/Fail + Certificate download
- **Rewards System** — Points, badges, leaderboard

---

## 🛠 Tech Stack (Prototype)
- React 18 + Vite
- Zero external UI libraries — pure inline styles
- Google Fonts: Inter + Plus Jakarta Sans
